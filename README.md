## CMSC733 - Alohomora (Phase 1)
### Steps to run the code
1) Open terminal and change the directory where the code is.
2) In the code, you will have to define the paths for the image, canny and sobel baselines.
3) The code is set to only view the outputs by default. Also, you can uncomment the line 292 to enable all filters.
4) Please add the 'BSD500' folder where the 'Wrapper.py' file is to successfully run the code.
In the terminal window, type the following to run the code.
```
python3 Wrapper.py
```
